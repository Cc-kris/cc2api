选择左侧客户端查看配置。所有入口都使用同一份 config.toml，API 密钥填写在 experimental_bearer_token。

Codex APP 配置说明

统一使用 config.toml

这个客户端入口统一使用同一份 config.toml。只需要修改配置目录中的 config.toml，并把 API 密钥填写到 experimental_bearer_token。

系统配置文件路径处理方式
macOS / Linux~/.codex/config.toml创建或覆盖 config.toml
Windows%userprofile%\.codex\config.toml创建或覆盖 config.toml

config.toml 内容

model_reasoning_effort = "high"

approval_policy = "never"
model = "gpt-5.5"
model_auto_compact_token_limit = 945000
model_context_window = 1050000
model_provider = "ccai"
personality = "pragmatic"
sandbox_mode = "danger-full-access"
tool_output_token_limit = 6000

[model_providers.ccai]
base_url = "https://cc-ai.xyz"
name = "OpenAI"
experimental_bearer_token = "填写API秘钥"
wire_api = "responses"
requires_openai_auth = false
supports_websockets = true

[features]
goals = true
multi_agent = true
image_generation = true

child_agents_md = true
hooks = true
suppress_unstable_features_warning = true
memories = true

完成检查

检查项正确结果
模型gpt-5.5
Providerccai
Base URLhttps://cc-ai.xyz
密钥字段experimental_bearer_token 已填写真实 API 密钥
认证开关requires_openai_auth = false
WebSocketsupports_websockets = true

## 2. 任务拆分原则

1. 每个任务只交付一个明确能力。
2. 每个任务都有独立验收标准。
3. 数据迁移、接口、前端、测试分开拆分。
4. 高级缓存和语义缓存默认不阻塞阶段 1-3 主版本。
5. 所有任务必须满足权限、脱敏、异常降级要求。

## 2. 总览结论

`0.2.43` 已具备基础图片桥接能力，但在“请求生命周期”和“重复请求语义”上不完整。`0.2.44` 的目标是把所有图片生成场景按真实新请求、自动重试、处理中、成功回放、失败退避、多步骤编辑六类闭环处理。

## 3. 场景核对矩阵

| 编号 | 场景 | `0.2.43` 状态 | 缺口 | `0.2.44` 落地结果 | 验收方式 |
|---|---|---|---|---|---|
| S01 | 纯文本 Responses | 满足 | 无 | 不进入图片桥接 | 纯文本请求不触发 Images 上游 |
| S02 | 顶层图片模型生成 | 基本满足 | 重复请求语义不完整 | 生成请求进入 `/images/generations`，重复按请求 ID 判断 | 单测验证上游路径 |
| S03 | 显式图片工具生成 | 基本满足 | 同一 turn 内二次真实请求可能被压制 | 不同客户端请求 ID 重新生成 | 连续两次请求上游计数为 2 |
| S04 | 非流式图片生成 | 基本满足 | 成功重复无完整回放 | 成功重复返回完整 JSON output | 检查 `output[0].result` |
| S05 | 流式图片生成 | 不完全满足 | 重复返回空 completed | 成功重复返回完整 SSE 图片事件 | 检查 `response.output_item.done` |
| S06 | `n>1` 多图生成 | 基本满足 | 回放只保存计数，不保存结果 | 回放包含全部图片 | 检查 output 数量 |
| S07 | 输入图编辑 | 满足 | 需回归保护 | 继续进入 `/images/edits` | 检查上游路径与 images 字段 |
| S08 | 上一轮生图继续编辑 | 满足 | 需回归保护 | `image_generation_call.result` 转为编辑输入 | 多步骤单测 |
| S09 | mask 编辑 | 基本满足 | 需回归保护 | `input_image_mask` 继续传递 | 解析单测 |
| S10 | 自动重试成功回放 | 不满足 | 第二次被压制为空成功 | 不重复上游，完整回放 | 相同客户端请求 ID 单测 |
| S11 | 自动重试处理中 | 不满足 | 返回空 completed | 返回处理中语义和 `Retry-After` | 手工制造 processing 记录 |
| S12 | 同 turn 新请求 | 不满足 | turn 覆盖 client request，误判重复 | client request 优先，重新生成 | 不同 client ID 同 turn 单测 |
| S13 | 客户端断开 | 不满足 | 成功状态写库可能被取消 | 后台短超时保存结果 | canceled context 单测 |
| S14 | 上游请求失败 | 基本满足 | 状态写入可能被取消 | 后台短超时标记可重试 | 错误单测 |
| S15 | 上游空图片 | 基本满足 | 需确认不写成功 | 标记可重试并返回错误 | 空 data 单测 |
| S16 | 指纹冲突 | 部分满足 | 返回语义不够明确 | 返回冲突/失败，不伪成功 | 同请求 ID 不同 payload 单测 |
| S17 | 幂等存储不可用 | 满足 | 日志需保留 | 降级为普通请求 | 模拟 repo 错误 |
| S18 | OAuth 原生图片路径 | 不在本次桥接范围 | 需回归不影响 | 原逻辑保持 | 既有测试通过 |
| S19 | API Key 原生 Images 路径 | 不在本次桥接范围 | 需回归不影响 | 原逻辑保持 | 既有测试通过 |
| S20 | 权限与模型映射 | 基本满足 | 需保持账单模型/上游模型分离 | 不改变现有映射，仅保留回放元数据 | 既有使用量测试通过 |

## 4. 改造前后行为对比

| 行为点 | 改造前 | 改造后 |
|---|---|---|
| 幂等身份 | 优先 `turn_id` | 优先 `X-Client-Request-ID`，无客户端 ID 才用 `turn_id` |
| 成功状态写入 | 使用请求上下文 | 使用后台短超时上下文 |
| 成功记录内容 | 仅保存状态和图片数量 | 保存完整可回放响应 |
| 成功重复请求 | 返回空 completed | 回放完整图片结果 |
| 处理中重复请求 | 返回空 completed | 返回处理中语义和 `Retry-After` |
| 用户再次点击生成 | 可能被同 turn 压制 | 不同客户端请求 ID 重新生成 |
| 客户端断开 | 可能状态写入失败 | 结果保存不依赖客户端连接 |
| 流式回放 | 无完整回放 | 回放完整 SSE |
| 错误退避 | 状态写入可能被取消 | 状态写入独立于请求取消 |

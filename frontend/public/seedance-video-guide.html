Seedace 视频调用说明

CCAI 提供 Seedance 2.0 视频生成接口，接口固定使用 /v1/video/generations。视频生成是异步任务：先提交拿 task_id，再轮询任务状态，完成后读取视频链接。

Base URL

https://cc-ai.xyz/v1

认证方式

Authorization: Bearer 你的key

国内seedace

Seedance 2.0 全能视频生成，支持文生、图生、多图组合、首帧、首尾帧、参考视频、参考音频。 走 /v1/video/generations，不要使用聊天接口。

需先在「API 密钥」页创建一把「国内seedace」档的 key。该 key 专用于 seedance-2.0-720 / seedance-2.0-1080 模型。

模型与价格(按视频秒数)

模型 分辨率 原价 ¥/秒 10 秒 / 15 秒原价
seedance-2.0-720 720P ¥1.00 ¥10.00 / ¥15.00
seedance-2.0-1080 1080P ¥1.20 ¥12.00 / ¥18.00

提交任务(文生视频)

bash
curl https://cc-ai.xyz/v1/video/generations \
  -H "Authorization: Bearer 你的key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "seedance-2.0-720",
    "prompt": "霓虹雨夜街头的电影感跟拍镜头，缓慢推进，35mm 颗粒",
    "aspect_ratio": "16:9",
    "seconds": "10"
  }'

轮询直到完成

bash
curl https://cc-ai.xyz/v1/video/generations/task_xxx \
  -H "Authorization: Bearer 你的key"

务必轮询到 status 变 completed / SUCCESS 再取视频。生成中时 video_url 为空或不可用。

参数总表

参数必填说明
model必填seedance-2.0-720 / seedance-2.0-1080
prompt必填画面提示词；多素材时用 @Image1 / @Video1 / @Audio1 指代
aspect_ratio否16:9 / 9:16 / 1:1 / 4:3 / 3:4 / 21:9
seconds否时长字符串，当前支持 "10" / "15"，默认 10
image_url否单张参考图 URL 或 base64 dataURL
reference_image_urls否多张参考图数组，最多 9 张
reference_videos否参考视频数组，最多 3 个，总时长建议不超过 15 秒
audio_url否参考音频；使用音频时需同时带至少一张参考图

玩法示例

bash
curl https://cc-ai.xyz/v1/video/generations \
  -H "Authorization: Bearer 你的key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "seedance-2.0-720",
    "prompt": "@Image1 角色随 @Audio1 的节奏起舞，运镜参考 @Video1",
    "seconds": "15",
    "reference_image_urls": ["https://你的图床/role.jpg"],
    "reference_videos": ["https://你的视频/camera.mp4"],
    "audio_url": "https://你的音频/track.mp3"
  }'

Python 完整示例

python
import time
import requests

BASE = "https://cc-ai.xyz/v1/video/generations"
H = {"Authorization": "Bearer 你的key", "Content-Type": "application/json"}

task = requests.post(BASE, headers=H, json={
    "model": "seedance-2.0-720",
    "prompt": "一只橘猫在窗台上伸懒腰，慢镜头，暖色调",
    "aspect_ratio": "16:9",
    "seconds": "10",
}).json()

tid = task.get("task_id") or task.get("id")

for _ in range(120):
    result = requests.get(f"{BASE}/{tid}", headers=H).json()
    status = str(result.get("data", {}).get("status") or result.get("status") or "").lower()
    if status in ("completed", "success", "failed", "failure"):
        print(status, result)
        break
    time.sleep(8)

国际seedace

即梦 Seedance 2.0 官方满血源，质量优先，与上方国内 Seedace 是两套独立源与价格。 支持文生、图生、首尾帧、参考视频、参考音频，按视频秒数计费。

需先在「API 密钥」页创建一把「国际seedace」档的 key。该 key 专用于下列 dreamina-seedance-2-0-* 模型。

模型与价格(按视频秒数)

模型(文生 / 图生·首尾帧·音频用 -ref) 分辨率 文生原价 ¥/秒 带图(-ref)原价 ¥/秒
dreamina-seedance-2-0-480p[-ref]480P¥0.52¥0.33
dreamina-seedance-2-0-720p[-ref]720P¥1.12¥0.68
dreamina-seedance-2-0-1080p[-ref]1080P¥2.78¥1.69
dreamina-seedance-2-0-fast-480p[-ref]480P 快¥0.42¥0.24
dreamina-seedance-2-0-fast-720p[-ref]720P 快¥0.91¥0.53

纯文字用不带 -ref 的；带图 / 首尾帧 / 音频用带 -ref 的。duration 控制秒数，默认 4。

参数总表

参数适用说明
model必填上表模型名，决定分辨率和计费档
prompt必填画面描述
duration否秒数，默认 4；同义字段 seconds
aspect_ratio否16:9 / 9:16 / 1:1 / 4:3 / 3:4 / 21:9
image / images-ref参考图，单张或多张，最多 9 张；也兼容 image_url / reference_image_urls
first_frame / last_frame-ref首帧 / 尾帧图
reference_videos-ref参考视频数组，最多 3 个，单段建议不超过 15 秒
audio_url-ref参考音频，需配至少 1 张图
generate_audio全部是否生成 AI 声音，默认 true；传 false 得静音视频

Python 完整示例

python
import time
import requests

BASE = "https://cc-ai.xyz/v1"
KEY = "你的国际seedace key"
H = {"Authorization": f"Bearer {KEY}", "Content-Type": "application/json"}

task = requests.post(f"{BASE}/video/generations", headers=H, json={
    "model": "dreamina-seedance-2-0-720p-ref",
    "prompt": "镜头缓缓推进，画面动起来",
    "duration": 5,
    "image": "https://你的图床/photo.jpg",
}).json()

tid = task.get("task_id") or task.get("id")

for _ in range(120):
    result = requests.get(f"{BASE}/video/generations/{tid}", headers=H).json()
    status = str(result.get("data", {}).get("status") or result.get("status") or "").upper()
    if status in ("SUCCESS", "FAILURE", "FAILED"):
        print(status, result)
        break
    time.sleep(8)

常见问题

无可用渠道 / 模型不存在key 不是「国际seedace」档，或模型名拼写错误。
视频没有声音本节模型默认带声音；若传了 generate_audio:false 会静音。
参考图报错图太小会被上游拒，建议使用不低于 512px 的图片。
视频链接过段时间失效生成结果链接可能有有效期，拿到后及时转存。
